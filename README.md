TrailTracer
===========

Application permettant à l'utilisateur d'envoyer périodiquement (modifiable) un SMS à une ou plusieurs personnes de son 
choix ses coordonnées GPS. La notion de parcours n'est pas a écarter.

L'utilisateur pourra ensuite manuellement entrer les coordonéees dans son système de cartographie.
Pour les receveurs équipés du système Android, une application interceptera les SMS afin de présenter 
les informations de manière plus user-friendly.
